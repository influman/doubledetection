# Installation
Couplage de deux détecteurs de mouvements dans un seul détecteur virtuel.  
Objectif : limiter les fausses détections d'intrusion  
  

### Ajout du périphérique 
Cliquez sur "Configuration" / "Ajouter ou supprimer un périphérique" / "Store eedomus" / "Double Détection" / "Créer"  

![STEP1](https://i.imgur.com/TkR7YAd.png)


*Voici les différents champs à renseigner:*

* [Obligatoire] - Le nom de l'emplacement du 1er détecteur  
* [Obligatoire] - Le périphérique 1er Détecteur de mouvement
* [Obligatoire] - Le nom de l'emplacement du 2nd détecteur 
* [Obligatoire] - Le périphérique 2nd Détecteur de mouvement
  
Un capteur "Double Détection" sera alors disponible :  

![STEP4](https://i.imgur.com/FPgeWGD.png)
